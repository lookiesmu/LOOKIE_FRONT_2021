# Chap 02-4 연산자

2021.04.01

<br>

### 01. 산술 연산자

* +, -, *, /(몫만 취함), %(나머지만 취함)

<br>

### 02. 대입 연산자

* 좌변 = 우변 (우변 값이 좌면에 들어간다)
* a = 5 (o), 5 = a (x)

<br>

### 03. 복합 대입 연산자

* +=, -=, *=, /=, %=
* a += 10 -> a = a + 10

<br>

### 04. 증감 연산자

* ++, --
* a++, a--

<br>

### 05. 비교 연산자

* `>`, >=, <, <=, ==, !=
* True == True -> True
* True == False -> False
* True != Ture -> False
* True != False -> True

<br>

### 06. 논리 연산자

* &&, ||, !
* True && True -> True
* True || False -> True
* !True -> False
* !False -> True
